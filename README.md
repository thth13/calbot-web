# CalBot Landing

Отдельный Next.js лендинг для деплоя на Vercel.

## Локальный запуск

```bash
npm install
npm run dev
```

## Vercel

При импорте репозитория в Vercel укажи root directory:

```text
calbot-landing
```

Build command:

```text
npm run build
```

Output directory оставь пустым, Vercel определит Next.js автоматически.
